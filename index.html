Highlighted Projects

Problem Statement

University students are overwhelmed by too many class options, causing stress and poor course choices.

Affinity Diagram

We sorted varying questions and problems in an attempt to formulate quality solutions for the issue of students being overwhelmed with options for classes to take. These include the problems of format, target users, school usage, and implementation methods such as mobile apps, websites, and other aspects such as ease of use and inclusivity. This resulted in an idea for an AI-powered program that recommends classes to students based on their enjoyment and performance in other classes.

Wireframe Sketches

Created 3 simple wireframe sketches showcasing the utility of my app, "CourseCompass". This is done through showing things such as account creation, adding classes to get personalized recommendations using AI, and finally showing how school administration could easily and efficiently manage students' classes via their phones, with personalized recommendations for each student.

Prototype

Paper prototype demoing features of the AI-powered course recommendation app.

Software Engineering Semester Group Project: Language Learning App

This repository is the main repository of TheCoolCoders and our app project, "LoveLearningLangs". The goal of this project is to create an application that can be used by individuals to learn languages. Users will be able to create an account, store progress, choose between languages, work through a course list, and complete exercises.
